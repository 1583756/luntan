# 🎉 论坛体小说生成器 - 直接打开网址就能用！

## 🚀 一键部署到云端（推荐，完全免费）

### Linux/Mac用户：
```bash

```

```bat
deploy-to-render.bat
```

然后按照提示操作，几分钟内就能得到一个固定网址，直接分享给朋友！

## 🎊 立即开始！

### 最快的方式（1分钟）
1. 双击 `start.bat`（Windows）或运行 `./start.sh`（Linux/Mac）
2. 打开浏览器访问 http://localhost:5000
3. 输入主题，点击生成
4. 享受你的论坛体小说！

### 想要分享给朋友？
运行 `deploy-to-render.sh` 或 `deploy-to-render.bat`，几分钟内就能得到一个固定网址！

## 📱 手机上像App一样使用！

### PWA（推荐，最简单）

**iOS用户：**
1. Safari访问应用网址
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 在主屏幕看到"论坛小说"图标
5. 点击打开，全屏显示！

**Android用户：**
1. Chrome访问应用网址
2. 点击右上角菜单
3. 选择"添加到主屏幕"
4. 在主屏幕看到App图标
5. 点击打开使用！

✅ **特点**：
- 自定义图标
- 全屏显示
- 离线支持
- 无需安装

👉 **详细教程**: 查看 [移动应用使用指南.md](移动应用使用指南.md)

### 打包成原生APK

将Web应用打包成原生Android应用，可以独立安装运行。

**Windows用户**：
```bat
build-apk.bat
```

**Linux/Mac用户**：
```bash
./build-apk.sh
```

然后按照提示在Android Studio中构建APK。

✅ **特点**：
- 完全原生体验
- 独立安装
- 可上传到应用商店
- 无需浏览器

👉 **详细教程**: 查看 [APK打包指南.md](APK打包指南.md)
