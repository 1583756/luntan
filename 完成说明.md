## 🎯 你现在可以做什么？

### 1️⃣ 本地体验（1分钟）
```bash
# Windows
start.bat

# Linux/Mac
./start.sh
```
然后访问：http://localhost:5000

### 2️⃣ 云端部署（10分钟）
```bash
# Linux/Mac
./deploy-to-render.sh

# Windows
deploy-to-render.bat
```
几分钟内就能得到一个固定网址！

### 3️⃣ 生成二维码（随时）
```bash
# Windows
generate_qrcode.bat

# Linux/Mac
python generate_qrcode.py
```
输入你的网址，生成二维码分享！

## 📱 推荐使用流程

### 🏃‍♂️ 新手快速开始
1. 运行 `start.sh` 或 `start.bat`
2. 打开浏览器访问 http://localhost:5000
3. 点击示例标签（如"不体面的事"）
4. 点击"✨ 生成小说"
5. 等待10-30秒
6. 查看结果，复制或下载

### ☁️ 分享给朋友
1. 运行 `deploy-to-render.sh` 或 `deploy-to-render.bat`
2. 按提示在浏览器操作
3. 等待部署完成（5-10分钟）
4. 获取固定网址，分享给朋友
5. 朋友打开网址就能使用了！

### 📱 移动端使用
1. 先完成云端部署
2. 在手机浏览器打开网址
3. 输入主题，点击生成
4. 享受移动端体验！

## 🆘 遇到问题？

### 常见问题
1. **启动失败** - 检查Python版本和依赖安装
2. **生成慢** - 正常情况，需要10-30秒
3. **无法访问** - 检查端口是否被占用
4. **部署失败** - 查看 `云端部署指南.md`

### 获取帮助
1. 查看文档 - 根目录的 `.md` 文件
2. 查看日志 - 终端输出或平台日志
3. API文档 - http://localhost:5000/docs
