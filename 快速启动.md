# 🚀 快速启动指南

## 第一步：启动服务

### Linux/Mac 用户：
```bash
./start.sh
```

### Windows 用户：
```bat
start.bat
```

### 手动启动：
```bash
python src/main.py -m http -p 5000
```

## 第二步：打开浏览器

在浏览器中输入：

```
http://localhost:5000
```

或直接：

```
http://localhost:5000/assets/index.html
```

## 第三步：使用界面

1. **输入主题**
   - 在输入框中输入你的小说主题
   - 或点击下方的示例标签

2. **点击生成**
   - 点击"✨ 生成小说"按钮
   - 等待10-30秒

3. **查看结果**
   - 自动显示生成的小说
   - 可以滚动查看完整内容

4. **保存小说**
   - 📋 点击"复制"按钮复制到剪贴板
   - 💾 点击"下载"按钮保存为txt文件

## 🔧 常见操作

### 修改端口
```bash
python src/main.py -m http -p 8080
```

然后访问：`http://localhost:8080`

### 局域网访问
1. 查看本机IP：
   - Windows: `ipconfig`
   - Linux/Mac: `ifconfig` 或 `ip addr`

2. 在其他设备访问：
   ```
   http://你的IP:5000
   ```

### 外网访问（使用ngrok）
```bash
# 1. 安装ngrok
# 2. 启动服务
./start.sh

# 3. 新开终端
ngrok http 5000

# 4. 复制ngrok提供的地址分享给朋友
```

## 🆘 遇到问题？

### 1. 无法启动
- 检查Python版本（3.8+）
- 检查依赖是否安装：`pip install -r requirements.txt`

### 2. 无法访问
- 检查端口是否被占用
- 尝试其他端口：`python src/main.py -m http -p 8080`
- 检查防火墙设置

### 3. 生成失败
- 检查网络连接
- 查看日志：`tail -f /app/work/logs/bypass/app.log`
- 等待一段时间后重试

### 4. 页面空白
- 刷新浏览器（F5）
- 清除浏览器缓存
- 尝试其他浏览器
