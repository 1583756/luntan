## 1. 准备工作

### 1.1 确认文件

你的项目文件夹应该包含以下文件：

**必需文件：**
```
论坛体小说生成器/
├── src/                          # 工作流代码
│   ├── main.py
│   ├── graphs/
│   └── ...
├── config/                       # 配置文件
├── assets/                       # Web界面
│   ├── index.html
│   ├── manifest.json
│   └── ...
├── start.sh                      # 启动脚本
├── start.bat                     # 启动脚本
├── deploy-to-render.sh           # 部署脚本
├── deploy-to-render.bat          # 部署脚本
└── requirements.txt              # 依赖文件
```

**检查方法：**
1. 打开项目文件夹
2. 确认上述文件都存在
3. 如果缺少，重新生成或联系支持

---

### 1.2 检查Python

**Windows:**

1. 按 `Win + R` 键
2. 输入 `cmd`，按回车
3. 输入：`python --version`
4. 应该看到类似：`Python 3.11.0`

**如果没有Python:**
1. 访问：https://www.python.org/downloads/
2. 下载最新版本（3.8或更高）
3. 运行安装程序
4. ⚠️ **重要**：勾选 "Add Python to PATH"
5. 点击 "Install Now"
6. 安装完成后重启命令提示符

**Mac/Linux:**

打开终端，输入：
```bash
python3 --version
```

**如果没有Python:**
```bash
# Mac
brew install python3

# Linux (Ubuntu/Debian)
sudo apt install python3 python3-pip
```

---

### 1.3 安装Python依赖

**Windows:**

1. 打开命令提示符（Win+R → cmd → 回车）
2. 进入项目文件夹：
   ```
   cd C:\你的项目路径
   ```
   示例：
   ```
   cd C:\Users\你的用户名\Desktop\forum-novel-generator
   ```
3. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
4. 等待安装完成

**Mac/Linux:**

1. 打开终端
2. 进入项目文件夹：
   ```bash
   cd /path/to/your/project
   ```
3. 安装依赖：
   ```bash
   pip3 install -r requirements.txt
   ```
4. 等待安装完成

**Windows用户：**

**方法1：双击启动**
1. 在项目文件夹找到 `start.bat`
2. 双击运行
3. 会看到一个黑色窗口（命令提示符）
4. 看到类似输出表示成功：
   ```
   INFO:     Started server process
   INFO:     Waiting for application startup.
   INFO:     Application startup complete.
   INFO:     Uvicorn running on http://0.0.0.0:5000
   ```

**方法2：命令行启动**
1. 打开命令提示符
2. 进入项目文件夹
3. 输入：
   ```
   start.bat
   ```

**Mac/Linux用户：**

1. 打开终端
2. 进入项目文件夹
3. 运行：
   ```bash
   ./start.sh
   ```

### 2.2 访问应用

1. 打开浏览器（Chrome、Edge、Firefox等）

### 2.4 局域网访问（手机使用）

**第1步：查看电脑IP**

**Windows:**
1. 打开命令提示符
2. 输入：
   ```
   ipconfig
   ```
3. 找到 "IPv4 地址"，如：
   ```
   IPv4 地址 . . . . . . . . . . . : 192.168.1.100
   ```
4. 记住这个地址（192.168.1.100）

**Mac:**
1. 打开终端
2. 输入：
   ```bash
   ifconfig
   ```
3. 找到 "en0" 下的 "inet"，如：
   ```
   inet 192.168.1.100
   ```
4. 记住这个地址

**Linux:**
```bash
ip addr show
```
或
```bash
ifconfig
```

**第2步：手机访问**
1. 确保手机和电脑连接**同一个WiFi**
2. 打开手机浏览器
3. 输入：
   ```
   http://192.168.1.100:5000
   ```
   （替换成你的实际IP）
4. 访问使用

**第3步：添加到主屏幕（可选）**

**iOS（iPhone/iPad）:**
1. Safari访问应用
2. 点击底部分享按钮（向上箭头）
3. 滑动找到"添加到主屏幕"
4. 点击确认
5. 在主屏幕看到"论坛小说"图标
6. 点击打开，全屏显示

**Android:**
1. Chrome访问应用
2. 点击右上角菜单（三个点）
3. 选择"添加到主屏幕"
4. 确认添加
5. 在主屏幕看到App图标
6. 点击打开使用

## 4. 打包APK

- ✅ 完全原生应用

- ✅ 更好的用户体验

---

1. 访问：https://nodejs.org
2. 下载LTS版本（推荐18.x或20.x）
3. 双击安装程序
4. 一路 "Next"
5. 安装完成后重启命令提示符

```
node -v
npm -v
```

```bash
brew install node
```

**Linux:**
```bash
sudo apt install nodejs npm
```

---

**下载安装：**
1. 访问：https://adoptium.net/
2. 选择版本17或更高
3. 下载安装包
4. 安装

**验证安装：**
```
java -version
```

---

**第3步：安装Android Studio**

**下载安装：**
1. 访问：https://developer.android.com/studio

### 4.5 构建Release APK（发布用）

如果你想要：
- 上传到应用商店
- 正式发布
- 分享给更多人

需要构建Release APK。

---

**第1步：生成密钥库**

在命令提示符中：

```
keytool -genkey -v -keystore forum-novel.jks -keyalg RSA -keysize 2048 -validity 10000 -alias forum-novel
```

**填写信息：**
- 输入密钥库密码（至少6个字符，记住这个密码）
- 重新输入密码
- 输入姓名
- 输入组织单位
- 输入组织名称
- 输入城市
- 输入省份
- 输入国家代码（CN）
- 确认信息

**保存文件：**
- 生成的文件：`forum-novel.jks`
- 记住密码！非常重要！

---

**第2步：在Android Studio生成Release APK**

1. 点击 "Build" → "Generate Signed Bundle / APK"
2. 选择 "APK"
3. 点击 "Next"
4. 点击 "Create new..."
5. 填写信息：
   - **Keystore file**: 选择 `forum-novel.jks`
   - **Password**: 输入密钥库密码
   - **Key alias**: `forum-novel`
   - **Key password**: 输入密码
6. 点击 "Next"
7. 选择 "release"
8. 选择签名版本：
   - V1 (Jar Signature): ✅ 勾选
   - V2 (Full APK Signature): ✅ 勾选
9. 点击 "Finish"
10. 等待构建完成

---

**第3步：获取Release APK**

位置：
```
forum-novel-apk\android\app\build\outputs\apk\release\app-release.apk
```

这个APK可以：
- 上传到应用商店
- 正式发布
- 分享给朋友

## 5. 常见问题

### 5.1 本地运行问题

**Q: start.bat运行后闪退？**
A:
1. 用命令提示符运行，查看错误信息
2. 检查Python是否安装
3. 检查依赖是否安装完整

**Q: 无法访问localhost:5000？**
A:
1. 检查黑色窗口是否还在运行
2. 确认端口5000没被占用
3. 尝试重启服务

**Q: 生成小说失败？**
A:
1. 检查网络连接
2. 查看错误信息
3. 确认主题输入正确

### 5.2 局域网访问问题

**Q: 手机无法访问？**
A:
1. 确认手机和电脑在同一WiFi
2. 检查电脑防火墙设置
3. 确认IP地址正确

**Q: 查不到IP地址？**
A:
- Windows: `ipconfig`
- Mac: `ifconfig`
- Linux: `ip addr show`

### 5.3 云端部署问题

**Q: GitHub推送失败？**
A:
1. 检查仓库地址是否正确
2. 确认GitHub账号密码
3. 检查网络连接

**Q: Render部署失败？**
A:
1. 查看Render控制台日志
2. 检查Dockerfile是否正确
3. 检查环境变量配置

**Q: 部署后无法访问？**

3. 清除浏览器缓存重试

**Q: 应用休眠后访问很慢？**
A:
- 这是正常现象
- 首次访问需要30秒启动
- 定期访问可以保持活跃

### 5.4 APK打包问题

**Q: Node.js安装失败？**
A:
1. 完全卸载Node.js
2. 重新下载安装
3. 勾选"Add to PATH"
4. 重启电脑

**Q: Gradle同步失败？**
A:
1. 检查网络连接
2. 点击 "File" → "Invalidate Caches / Restart"
3. 清理项目：`./gradlew clean`
4. 重新同步

**Q: 构建APK失败？**
A:
1. 查看错误日志
2. 清理项目：Build → Clean Project
3. 重新构建
4. 确认JDK版本正确

**Q: 安装APK失败？**
A:
1. 确认开启"允许安装未知来源应用"
2. 检查APK文件是否完整
3. 尝试重新传输APK

**Q: APK无法联网？**
A:
1. 检查capacitor.config.json中的URL
2. 确认网络权限已添加
3. 重新构建APK
