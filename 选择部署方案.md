# 🚀 快速部署指南 - 选择最适合你的方案

## 🤔 我想...

### 🎯 我想要最简单的方式，几分钟就能让别人访问

**→ 选择 ngrok（临时测试）**

```bash
# 步骤1：启动服务
./start.sh

# 步骤2：新开终端，映射端口
ngrok http 5000

# 步骤3：复制 ngrok 提供的网址
#    如：https://abc123.ngrok.io

# 步骤4：分享给朋友，他们直接打开就能用了！
```

**适合**：临时测试、快速演示、分享给朋友
**成本**：免费
**时间**：3分钟
**缺点**：网址每次重启都会变

---

### 💰 我想要免费长期使用，网址固定

**→ 选择 Render（推荐！）**

```bash
# 步骤1：运行一键部署脚本
./deploy-to-render.sh

# 步骤2：按提示在浏览器操作
# - 访问 https://dashboard.render.com/
# - 创建 Web Service
# - 等待部署完成

# 步骤3：访问固定网址
#    如：https://forum-novel-generator.onrender.com
```

**适合**：长期使用、固定网址、稳定运行
**成本**：免费
**时间**：10分钟
**优点**：自动HTTPS、自动重启、网址固定

---

### 🏢 我是企业用户，需要稳定可靠

**→ 选择云服务器**

```bash
# 步骤1：购买云服务器
# - 阿里云、腾讯云、华为云等

# 步骤2：连接服务器并部署
ssh root@你的服务器IP
# 按照云端部署指南.md操作

# 步骤3：访问
http://你的服务器IP:5000
```

**适合**：企业使用、高并发、完全控制
**成本**：$5+/月
**时间**：30分钟
**优点**：完全控制、高性能、可扩展
