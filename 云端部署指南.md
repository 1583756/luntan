# 论坛体小说生成器 - 云端部署指南

## 🌐 直接打开网址使用

### 方案1：部署到 Render（推荐，完全免费）

#### 步骤1：准备代码

确保你的项目包含以下文件：
- `src/` 目录
- `config/` 目录
- `requirements.txt`
- `Dockerfile`（我会帮你创建）

#### 步骤2：创建 Render 账号

1. 访问 https://render.com
2. 点击 "Sign Up" 注册账号
3. 使用 GitHub 账号登录（推荐）

#### 步骤3：创建 GitHub 仓库

1. 把你的项目推送到 GitHub
2. 仓库设为 Public（免费套餐要求）

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```

#### 步骤4：在 Render 部署

1. 登录 Render
2. 点击 "New" → "Web Service"
3. 连接你的 GitHub 仓库
4. 填写配置：

**Build & Deploy**:
- Name: `forum-novel-generator`
- Runtime: `Docker`
- Root Directory: `.`

**Dockerfile Path**: `Dockerfile`

**Environment**（环境变量）:
- `COZE_WORKSPACE_PATH`: `/app`

**Plan**: 选择 **Free**（免费）

5. 点击 "Create Web Service"
6. 等待 5-10 分钟部署完成
7. 部署完成后会得到一个公网地址，如：
   ```
   https://forum-novel-generator.onrender.com
   ```

#### 步骤5：访问你的应用

直接在浏览器打开 Render 提供的网址，就可以直接使用了！

### 方案2：部署到 Railway（完全免费）

#### 步骤1：注册 Railway

1. 访问 https://railway.app
2. 点击 "Start a new project"
3. 使用 GitHub 账号登录

#### 步骤2：连接仓库

1. 选择 "Deploy from GitHub repo"
2. 选择你的仓库
3. Railway 会自动检测到 Dockerfile

#### 步骤3：配置环境变量

添加环境变量：
- `COZE_WORKSPACE_PATH`: `/app`

#### 步骤4：部署

点击 "Deploy"，等待 3-5 分钟

#### 步骤5：获取访问地址

Railway 会自动生成一个公网地址

---

### 方案3：使用 ngrok（临时测试，最简单）

如果你只是想快速测试给别人看，使用 ngrok 最简单：

#### 步骤1：安装 ngrok

- Windows: https://ngrok.com/download
- Mac: `brew install ngrok`
- Linux: 从官网下载

#### 步骤2：注册 ngrok

1. 访问 https://ngrok.com
2. 注册账号（免费）
3. 获取 authtoken

```bash
ngrok config add-authtoken 你的token
```

#### 步骤3：启动服务

```bash
./start.sh
```

#### 步骤4：映射端口

新开一个终端：

```bash
ngrok http 5000
```

#### 步骤5：分享链接

ngrok 会显示一个公网地址，如：
```
https://abc123.ngrok.io
```

把这个地址分享给朋友，他们就可以直接访问了！

**注意**：ngrok 的地址每次重启都会改变，适合临时测试。

### 方案4：部署到自己的服务器

#### 步骤1：购买云服务器

推荐平台：
- 阿里云（新用户有优惠）
- 腾讯云
- 华为云
- AWS（有免费套餐）
- DigitalOcean（$5/月）

#### 步骤2：连接服务器

```bash
ssh root@你的服务器IP
```

#### 步骤3：安装环境

```bash
# 安装 Python
apt update
apt install python3 python3-pip -y

# 安装依赖
pip3 install -r requirements.txt
```

#### 步骤4：上传代码

```bash
# 在本地
scp -r /path/to/project root@服务器IP:/root/

# 或使用 Git
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名
pip3 install -r requirements.txt
```

#### 步骤5：启动服务

```bash
# 方式1：前台运行
python3 src/main.py -m http -p 5000

# 方式2：后台运行
nohup python3 src/main.py -m http -p 5000 > app.log 2>&1 &

# 方式3：使用 PM2（推荐）
npm install -g pm2
pm2 start "python3 src/main.py -m http -p 5000" --name forum-novel
pm2 save
pm2 startup
```

#### 步骤6：开放端口

```bash
# Ubuntu/Debian
ufw allow 5000

# CentOS/RHEL
firewall-cmd --permanent --add-port=5000/tcp
firewall-cmd --reload
```

#### 步骤7：访问应用

```
http://你的服务器IP:5000
```

#### 步骤8：配置域名（可选）

1. 购买域名
2. 配置 DNS 解析到服务器IP
3. 使用 Nginx 反向代理

```nginx
server {
    listen 80;
    server_name 你的域名.com;

    location / {
        proxy_pass http://127.0.0.1:5000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
