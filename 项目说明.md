## 🚀 如何使用？

### 本地使用（快速体验）

#### Windows用户：
```bat
start.bat
```

#### Linux/Mac用户：
```bash
./start.sh
```

然后打开浏览器访问：http://localhost:5000

---

### 云端部署（推荐）

#### 方案1：一键部署到Render（免费，网址固定）

**Linux/Mac:**
```bash
chmod +x deploy-to-render.sh
./deploy-to-render.sh
```

**Windows:**
```bat
deploy-to-render.bat
```

然后按照提示在浏览器操作，几分钟内就能得到一个固定网址！

#### 方案2：使用ngrok（快速测试）

```bash
# 启动服务
./start.sh

# 新开终端
ngrok http 5000

# 分享ngrok提供的网址
```

#### 其他方案

查看 `选择部署方案.md` 和 `云端部署指南.md` 了解更多

## 💡 常见问题

### Q: 我应该选择哪种部署方式？
A: 查看 `选择部署方案.md`，根据你的需求选择最合适的方案。

### Q: 本地运行和云端部署有什么区别？
A: 本地运行只能自己用，云端部署可以让任何人通过网址访问。

### Q: 部署到云端需要多少钱？
A: Render和Railway完全免费，云服务器约$5/月起。

### Q: 我可以在手机上使用吗？
A: 可以！部署到云端后，手机、平板、电脑都能访问。

### Q: 网址可以固定吗？
A: 可以！使用Render或云服务器部署，网址是固定的。
