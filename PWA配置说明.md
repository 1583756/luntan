## 🚀 如何使用

### iOS（iPhone/iPad）

**第1步：部署应用**
- 使用云端部署或本地运行
- 获取访问地址

**第2步：Safari访问**
- 打开Safari浏览器
- 输入应用地址
- 等待页面加载完成

**第3步：添加到主屏幕**
1. 点击底部的"分享"按钮（向上箭头⬆️）
2. 向上滑动菜单
3. 找到"添加到主屏幕"选项
4. 点击

**第4步：确认添加**
1. 可以修改图标名称（默认"论坛小说"）
2. 点击右上角"添加"
3. 确认添加

**第5步：使用**
- 在主屏幕看到"论坛小说"图标
- 图标是紫色渐变，带书本图标
- 点击图标打开应用
- 应用全屏显示，没有Safari的地址栏

✅ **效果**：
- 独立的App图标
- 全屏显示
- 启动动画
- 像原生App一样

## 🔧 PWA配置详情

### manifest.json

```json
{
  "name": "论坛体小说生成器",
  "short_name": "论坛小说",
  "description": "基于LangGraph的论坛体小说生成工具",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#7c3aed",
  "theme_color": "#7c3aed",
  "orientation": "portrait"
}
```

### 关键配置说明

- **name**: 完整应用名称
- **short_name**: 简短名称（用于图标下方）
- **display**: "standalone" 表示独立窗口
- **theme_color**: 主题颜色（用于状态栏）
- **background_color**: 启动画面背景色

## 📱 PWA vs Web App

| 特性 | Web App | PWA |
|------|---------|-----|
| 访问方式 | 浏览器书签 | 主屏幕图标 |
| 图标 | ❌ 浏览器图标 | ✅ 自定义图标 |
| 显示方式 | 有地址栏 | 全屏显示 |
| 离线使用 | ❌ 完全不行 | ✅ 界面可用 |
| 启动速度 | 一般 | 快 |
| 更新方式 | 手动刷新 | 自动更新 |
| 存储占用 | 无 | 小（缓存） |

## 💡 常见问题

### Q: PWA需要安装吗？
A: 不需要传统安装，只需添加到主屏幕

### Q: 离线时能生成小说吗？
A: 界面可以打开，但生成小说需要网络连接（API调用）

### Q: 可以删除PWA吗？
A: 和删除普通App一样，长按图标选择删除

### Q: PWA会占用很多空间吗？
A: 只占用少量缓存空间，约几MB

### Q: PWA能接收通知吗？
A: 理论上可以，但目前未配置

### Q: iOS和Android都支持吗？
A: 完全支持！

### Q: 可以上架应用商店吗？
A: PWA不能上架，需要打包成原生APK
